# **App Name**: Code Insights

## Core Features:

- GitHub Repo Analysis: Connect to GitHub repositories to analyze code.
- AI-Generated Documentation: Generate documentation for the codebase using AI (Gemini + Genkit). The LLM uses reasoning as a tool to incorporate snippets of code in its output.
- Architecture Summaries: Provide high-level architecture summaries of the code.
- Interactive Insights Interface: Offer an interactive interface to explore the code, summaries, and documentation.

## Style Guidelines:

- Primary color: Electric purple (#892CDC) for a modern, techy vibe.
- Background color: Dark gray (#1E1E1E) to provide a dark theme.
- Accent color: Cyan (#00FFFF) for highlights and calls to action.
- Body and headline font: 'Inter' for a modern, neutral feel. It's a sans-serif font suitable for both headlines and body text.
- Use Lucide React icons for a clean, consistent look.
- Grid layout for the features section with hover animations.
- Subtle animations and micro-interactions throughout the site.
